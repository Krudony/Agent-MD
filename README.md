# Agent-MD

ที่เก็บนี้มีหน้าเว็บตัวอย่างสำหรับทดสอบการอัปโหลดไฟล์หนึ่งหน้า (`index.html`).

## วิธีเปิดดูหน้าเว็บ
1. ดาวน์โหลด/โคลนที่เก็บนี้ลงเครื่อง
2. ดับเบิลคลิกไฟล์ `index.html` เพื่อเปิดด้วยเว็บเบราว์เซอร์ (Chrome, Edge, Safari ฯลฯ)
   - หรือ รันคำสั่ง `python3 -m http.server 8000` จากโฟลเดอร์โปรเจกต์ แล้วเปิด `http://localhost:8000/index.html`

หน้าเว็บจะแสดงการ์ดให้อัปโหลดไฟล์ พร้อมแสดงชื่อไฟล์ที่เลือกและปุ่มจำลองการอัปโหลดสำหรับทดสอบ
